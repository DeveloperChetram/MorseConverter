Morse Code Converter Guide

Text to Morse Mode:

  • Only letters (A-Z) and numbers (0-9) are supported
  • Special characters are not allowed
  • Spaces between words will be converted to three spaces in morse code

Morse to Text Mode:

  • Only valid morse code characters (dots and dashes) are supported
  • Use single space to separate letters
  • Use three spaces to separate words
